There are many More API endpoints to try as an Authorized User:

    Create and Delete Post Request - HEADERS: Authorization Bearer TOKEN, Content-Type: application/json

  • Example: curl https://eman747-first-crud-api.herokuapp.com/api/posts -H 'Authorization: Bearer TOKEN'
  • GET          https://eman747-first-crud-api.herokuapp.com/api/posts -H 'Authorization: Bearer TOKEN'
  • CREATE https://eman747-first-crud-api.herokuapp.com/api/posts
  • DELETE https://eman747-first-crud-api.herokuapp.com/api/posts/:postId

    Create and Disable Personal User Account - HEADERS: Authorization Bearer TOKEN, Content-Type: application/json

  • DELETE https://eman747-first-crud-api.herokuapp.com/api/users/:userId
  • PATCH https://eman747-first-crud-api.herokuapp.com/api/users/:userId -X PATCH -H 'Authorization: Bearer TOKEN' -H 'Content-Type: application/json' -d '{"title": "Creativity is Beautiful", "content": "Creativity is intelligence having fun.", "tags": "#freshDev"}'
